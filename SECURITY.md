# Security Policy

## Reporting a Vulnerability

If you discover a security issue in RoleOS (the site at [roleos.fyi](https://roleos.fyi) or anything in this repo), please report it privately rather than opening a public issue.

**Email:** nikjain1588@gmail.com

Please include:
- A description of the issue and its impact
- Steps to reproduce
- Any relevant logs, URLs, or proof-of-concept

I'll acknowledge within 72 hours and aim to resolve confirmed issues promptly. Please give me reasonable time to ship a fix before public disclosure.

Thanks for helping keep RoleOS users safe.
